## Introduction

- A basic project to deploy vite project with react-router-dom to github-pages.

## Getting Started

- Supports basic auth
- Supports theme
- User Dashboard
- Supports auto deploy on push using `github actions`

![line]

## Developement

```sh
> cd client
> npm i
> npm run dev
```
